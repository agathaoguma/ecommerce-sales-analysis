# E-commerce Sales Performance Dashboard

## Overview
This project analyzes an e-commerce dataset to evaluate sales performance, customer behavior, and product trends. The dashboard provides insights into revenue, profit, and regional performance to support data-driven decisions.

## Tools Used
- Excel
- Power BI

## Key Objectives
- Identify top-performing products and categories
- Analyze monthly sales trends
- Evaluate customer segmentation
- Monitor profit margins across regions

## Key Insights
- Technology category generated the highest revenue of over 50%
- East Region records highest sales
- Repeat customers drove significant revenue contribution
- Discounts above 40% lower profit margin
